Como criar uma conta no JUFinanceiro ?

Siga estas etapas para criar uma conta no JuFinanceiro:

1. Acesse página oficial do JuFinanceiro: https://financeiro.homolog.janelaunica.com.br/#/login

2. Clique no link: https://financeiro.homolog.janelaunica.com.br/#/registro

3. Preencha os campos:

1. Nome de usuário.
2. e-mail.
3. Senha.
4. Clique em concordo com os termos e serviço e políticas e privacidade.


➡️ Quando estiver pronto, clique em CONTINUE.
➡️ Em seguinda informe se o tipo de conta é PESSOAL ou de NEGÓCIO:
➡️ Quando estiver pronto, clique em CRIAR CONTA.
➡️ Enviaremos um link com instruções para verificar seu e-mail. Então, está tudo pronto.
